# 📊 Interactive Sales Dashboard

## 📌 Project Overview

The Interactive Sales Dashboard is a Python-based data visualization project that analyzes sales data and presents meaningful business insights using Seaborn, Matplotlib, and Plotly.

The dashboard helps users understand sales performance, product-wise revenue, regional sales distribution, pricing trends, and correlations through both static and interactive visualizations.

---

## 🎯 Objectives

- Analyze sales data efficiently.
- Create statistical visualizations using Seaborn.
- Build interactive charts using Plotly.
- Generate business insights.
- Save visualizations automatically.
- Create a professional dashboard for decision-making.

## 📊 Dashboard Features

### 1. Line Chart
Displays sales trends over time.

### 2. Bar Chart
Compares total sales for different products.

### 3. Scatter Plot
Shows the relationship between Price and Total Sales.

### 4. Box Plot
Displays price distribution for each product.

### 5. Heatmap
Shows correlation among Quantity, Price, and Total Sales.

### 6. Plotly Interactive Charts
- Interactive Bar Chart
- Interactive Line Chart
- Interactive Pie Chart

## ⚙ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Interactive-Sales-Dashboard.git
```

Move into project folder:

```bash
cd Interactive-Sales-Dashboard
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶ Run the Project

Python File

```bash
python dashboard.py
```

Jupyter Notebook

```bash
jupyter notebook dashboard.ipynb
```
